# PlayGround1

A tiny playground project for experimenting with HTML, CSS, and JavaScript.

## What it does

This project lets you type in a color value and apply it to the page background.

You can enter:
- a CSS color name like `tomato`
- a hex color like `#00aaff`

There is also a random button that picks a random color for you.

## Purpose

This repository is a simple practice project for:
- working with DOM elements
- handling user input
- updating styles with JavaScript
- experimenting with small UI ideas

## How to use it

1. Open `index.html` in your browser.
2. Type a color value into the input box.
3. Click **Apply** or press **Enter**.
4. Try **Random** to generate a color automatically.

## Files

- `index.html` — main page with the structure, styles, and script
